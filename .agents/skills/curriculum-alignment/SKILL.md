# OMF Curriculum Alignment Guide

Follow these guidelines to map concept codes to regional learning frameworks:

## 1. Mappings Layout
Align new stages inside the appropriate mapping file:
- `curriculum/australia-v9-map.md`: Map to Foundation (AC9MFNxx) and Year 1-6 outcomes.
- `curriculum/common-core-map.md`: Map to Kindergarten (K.CC.x, K.OA.x) and Grade 1-6 CCSS domains.
- `curriculum/global-primary-map.md`: General age-progression synthesis.

## 2. Code Correlation
Match concept targets to correct standard codes:
- **One-to-One Correspondence**: CCSS.MATH.CONTENT.K.CC.B.4 | AC9MFN01
- **Comparison & Quantities**: CCSS.MATH.CONTENT.K.CC.C.6 | AC9MFN03
- **Number Line Ordering**: CCSS.MATH.CONTENT.K.CC.C.7 | AC9MFN03
- **Part-Whole Bonds**: CCSS.MATH.CONTENT.K.OA.A.3 | AC9MFN04
- **Teen Place Value**: CCSS.MATH.CONTENT.1.NBT.B.2 | AC9M1N01

## 3. Metadata Reference
Always list the primary curriculum standard codes inside the `Curriculum mapping` notes section of the `parent-guide.md` file.
