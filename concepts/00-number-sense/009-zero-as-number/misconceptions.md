# Common Misconceptions: Zero as a Number

Zero represents the count of an empty collection:

---

## 1. Zero is "Nothing" (Not a Number)
* **The Mistake**: The child skips 0 when writing lists or doesn't count it as a number in sequences, starting their number lines or rulers directly with 1.
* **The Conceptual Break**: Treating numbers only as physical things they can see. If they can't see anything, they think no number can represent it.
* **How to help**: Use labels. Label three plates: "1 cookie", "2 cookies", "0 cookies" (empty plate), showing that 0 is a valid label/number.

---

## 2. Click-to-Answer Impulse
* **The Mistake**: Handed a question "Put 0 apples in the basket", the child automatically drags items in anyway because they feel a task requires action.
* **The Conceptual Break**: Associating activity sheets with "doing" rather than representing the value.
* **How to help**: Point out the prompt and ask: "What does zero mean? It means we don't need to put anything in there! Let's hit check."
