# Educator Guide: Misconceptions in Place Value

When learning two-digit place value (tens and ones), children face several conceptual hurdles.

---

## 1. Digit Reversals (e.g. writing 32 for 23)
* **The Mistake**: Shown 2 tens and 3 ones, the child writes the digit representing the ones first, resulting in **32** instead of **23**.
* **Cognitive Break**: The child counts the ones first (3) and the tens second (2), and writes them in the order they counted, without realizing that digit positions are fixed by column value (Tens on the left, Ones on the right).
* **Correction Strategy**:
  - Use a visual Place Value Mat with labeled columns: **Tens** | **Ones**.
  - Have them place the rods in the Tens column and the blocks in the Ones column.
  - Write the digits directly below the columns so they physically see that the number of rods is written on the left.

---

## 2. Counting Rods as Units of One
* **The Mistake**: Shown 2 rods and 4 blocks, the child counts all shapes as ones, stating there are **6** objects total.
* **Cognitive Break**: The child lacks the concept of *unitizing*—the understanding that a single physical object (a rod) can represent a group of ten smaller objects.
* **Correction Strategy**:
  - Take a ten rod and have the child lay 10 individual unit blocks side-by-side on top of it.
  - Show that they cover the exact same space.
  - Practice counting: *"One ten (point to rod), two tens (point to rod)... 10, 20!"*
